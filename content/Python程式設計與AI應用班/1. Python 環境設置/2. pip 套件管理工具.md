## ㄧ. 什麼是 pip 和為什麼需要他？

`pip` 是一個 `Python` 安裝管理套件工具，如果你今天要使用任何與 `Python` 有關的套件，基本上都會用到 `pip` 來安裝或是更新 `Python` 相關套件。

## 二. 如何安裝 pip ?

1. 安裝 pip 前請留意
   目前 `Python 3` 的 `3.4` 版本以上 或是 `Python 2` 的 `2.7.9` 版本以上的 `Python` 都已經自帶 pip， 所以如果不是 `Python` 版本過舊的話，基本上不用需要安裝 `pip`。

   想要查詢目前使用的 `Python` 版本的話，在終端機中輸入以下指令，就可以知道目前的`Python` 版本囉：

   ```python
   python --version

   Python 3.7.7
   ```

2. Mac 的朋友安裝 pip 看這邊
   - 到 [pip document 官方網站](https://pip.pypa.io/en/stable/installing/) 建議的載點下載 `pip` 安裝檔
   - 啟動剛剛下載好的`get-pip.py`檔案，就安裝完成囉！

     ```python
     python get-pip.py
     ```
   - 如何查看`pip`目前版本?
  
     ```python
     python -m pip --version
     ```

   - 如何更新 pip 到最新版本?

     ```python
     python -m pip install --upgrade pip
     ```

3. Windows 的朋友安裝 pip 看這邊

   - 到 [pip document 官方網站](https://pip.pypa.io/en/stable/installing/) 建議的載點下載 pip 安裝檔

     ```python
     curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
     ```

   - 啟動剛剛下載好的 `get-pip.py` 檔案，就安裝完成囉！

     ```python
     py get-pip.py
     ```

   - 如何查看 pip 目前版本?

     ```python
     py -m pip --version   
     ```

   - 如何更新 pip 到最新版本

     ```python
     py -m pip install -U pip
     ``` 

4. Linux 的朋友安裝 pip 看這邊

   ```bash
   apt-get install python-pip
   ```

## 三. pip 指令全記錄

1. pip 安裝套件
   可以將 `<package-name>` 換成任何想安裝的 Python 套件

   ```bash
   pip install <package-name>
   ```

   加上 `<version>`參數，可以指定安裝的套件版本

   ```bash
   pip install <package-name>==<version>
   ```
   
2. pip 更新套件
   使用 `–upgrade` 或 `-U` 來更新套件至最新版本

   ```bash
   pip install --upgrade <package-name>
   pip install -U <package-name>
   ```
   
3. pip 移除安裝過的套件
   使用 uninstall 參數來移除安裝過的 python 套件

   ```bash
   pip uninstall <package-name>
   ```

4. 查看目前安裝過的 python 套件清單
   使用 list 參數會展示所有安裝過的套件和版本
   
   ```bash
   pip list 
   > Package                  Version
   > ------------------------ ---------
   > aiohttp                  3.8.1
   > aiosignal                1.2.0
   > argcomplete              1.12.3
   ```
  
   也可以針對特定套件顯示更詳細的套件資訊

 ```bash
 $ pip show <package-name>
 
 > Name: toml
 > Version: 0.10.2
 > Summary: Python Library for Tom's Obvious, Minimal Language
 > Home-page: https://github.com/uiri/toml
 > Author: William Pearson
 > Author-email: uiri@xqz.ca
 > License: MIT
 > Location: /usr/local/lib/python3.9/site-packages
 > Requires:
 > Required-by: pre-commit
 ```

## 四. 如何使用 pip 一次安裝多個套件

每次換環境就需要重新安裝套件，輸入指令重新安裝很麻煩，所以我們通常都會將要安裝的套件寫在 txt 檔案裡面，只需要 install 這個文字檔就可以一次安裝所有需要的套件囉！

1. pip 安裝 requirements.txt 內的清單：

   ```bash
   pip install -r requirements.txt
   ```
   
2. 將安裝過的套件建立成 requirements.txt 文件清單：

   ```bash
   pip freeze > requirements.txt
   ```
